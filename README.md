# PassOP - Password Manager

PassOP is a full-stack password management application built with React, Tailwind CSS, Express.js, and MongoDB. It allows users to securely store and manage website credentials through a clean and responsive interface.

## Features

- Create, view, edit, and delete passwords
- Copy credentials to clipboard
- Toggle password visibility
- Responsive UI built with Tailwind CSS
- Toast notifications using React Toastify
- Animated icons using Lordicon
- MongoDB database integration
- RESTful API with Express.js
- Persistent credential storage

## Tech Stack

- React.js
- Vite
- Tailwind CSS
- Node.js
- Express.js
- MongoDB
